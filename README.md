Nona Beauty - Ready-to-publish website (static)

This repository contains the full static website for Nona Beauty (built from Store.zip).

How to publish on GitHub Pages (automatic on push to main):
1. Create a new public repository named `Nona-beauty` on GitHub.
2. Push these files to the `main` branch:
   git init
   git add .
   git commit -m "Initial import of Nona Beauty site"
   git branch -M main
   git remote add origin https://github.com/YourUsername/Nona-beauty.git
   git push -u origin main
3. The GitHub Actions workflow will automatically publish the repository root to GitHub Pages using peaceiris/actions-gh-pages.
4. The site will be available at: https://YourUsername.github.io/Nona-beauty/
